# README #

### Hop ###
* Name (string)
* Origin (string)
* Type (exclusive)
    * bittering
    * aroma
    * both
* Form (exclusive)
    * pellet
    * plug
    * leaf
* Acids (exclusive)
    * Alpha (float)
    * Beta (float)
* Hop Storing Index (HSI) (float)
* Notes (string)

### Grain ###
* Name (string)
* Origin (string)
* Type (exclusive)
    * Malt
    * Extract
    * Dry extract
* Color (SRM) (integer)
* Yield (float)
* FG/CG (fine grain / coarse grain) (float)
* Moisture (float)
* DP (siastatic power) (float)
* Proteins (float)
* Max in batch (float)
* Notes (string) (float)
